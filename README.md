# myschools
take exam questions using the scraper method

## example code
```python
class Stu:
    def __init__(self, teks: str):
        self.teks = teks
    def _log(self, *args):
        return "".join(args)
    def runner(self):
        self._log(list(self.teks)

class console(Stu):
  def __init__(self, teks):
      self.teks = teks
  def main(self):
      print(self.runner(self.teks)

cwo = console("Hello world")
cwo.main()
```
